Play with letters — become a keyboard DJ

Press any letter key (A–Z) to hear the sound mapped to that letter. Type sentences into the text box to queue them as a sequence and add reverb or delay.

Start Playing

How to

  1. Open Sampler to start typing.
  2. Type sentences in the text area (on Sampler page) to play sounds.
  3. Adjust effects using Tutorial keys.
