Learning to DJ (Tutorial)

Press any letter key (A–Z) to hear the sound mapped to that letter. Type sentences into the text box to queue them as a sequence and add reverb or delay.

Start Playing

HI! Welcome to the Tutorial for KeyBeat


This tutorial will guide you through the features of KeyBeat and help you get started with creating music using your keyboard.

Lets start with typing in the Sampler page. Press any letter key (A-Z) to hear the sound mapped to that letter.

Type sentences into the text box and add a period (.) at the end to loop the sentence continuously. To stop the loop, simply delete the sentence from the text box.


Adjust effects using the following keywords in your sentences:

  • volumeX - Sets the volume level (0-10)
  • reverbX - Sets the reverb level (0-10)
  • delayX - Sets the delay level (0-10)

For example, typing "reverb" will give you a reverb level of 0 while "reverbbbbbb" will set it to 5.

Now that you know the basics, start experimenting with different sentences and effects to create your own unique songs!
